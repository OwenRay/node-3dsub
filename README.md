Srt-to-ass
=========

Convert subtitles (SRT) to Advanced Substation Alpha (ASS)

## Installation

  npm install srt-to-ass

## Usage

  var sub = require('srt-to-ass');
  
  sub.convert('stubs/bighero.srt', 'bighero.ass');
